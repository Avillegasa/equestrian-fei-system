# Sistema de Gestión de Competencias Ecuestres FEI

Sistema completo para la gestión y calificación de competencias ecuestres según los estándares oficiales de la Federación Ecuestre Internacional (FEI).

## 🏆 Características Principales

- ✅ Sistema de calificación FEI oficial (3 celdas)
- ✅ Rankings en tiempo real con WebSockets
- ✅ Funcionalidad offline para ubicaciones remotas
- ✅ Gestión completa de competencias y participantes
- ✅ Auditoría completa de calificaciones
- ✅ Interfaces responsivas para jueces y espectadores

## 🚀 Inicio Rápido

### Pre-requisitos

- Python 3.11+
- Node.js 18+
- PostgreSQL 14+
- Redis 6+
